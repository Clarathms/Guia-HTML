Formatação dos Textos:

Para Background:

Se usar o CSS na propria pagina, usar a tag style e fazer a partir dela (como eu fiz no head)

Principais Formatações:

Negrito: A tag é STRONG, NAO BOLD

Itálico: usa a tag I ou EM

OBS: Mudar assim é feito por html, mas é mais recomendado usar o CSS para fazer essas formatações

Texto Marcado:

A tag é MARK, para mudar a cor amarela base da marcação, usa o CSS

OBS: Por exemplo assim O CSS foi usado dentro da tag para mudar a cor, mas essa cor só fica nessa tag, se usar outro mark a cor vai ser amarela (cor habitual)
Mas se quiser mudar a cor de todos os marks usados nessa pag, alterar no CSS no head, dentro de Style com
mark {background-color:cor;}

Tamanho dos textos:

Texto Grande: A tag é BIG, mas está caindo em desuso

Texto Pequeno: A tag é SMALL, geralmente usado para "ler as letras pequenas" de um contrato por ex

Texto Deletado e Inserido

Deletado: Quando pode ser lido mas não deve ser considerado. A tag é DEL

Inserido: Para dar ênfase. A tag é INS

Texto Sobrescrito e Subscrito:

Sobrescrito: Escrever coisas do tipo X2 + 3 . A tag é SUP

Subscrito: Escrever coisas do tipo H2O . A tag é SUB
